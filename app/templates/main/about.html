О проекте

Веб-приложение предназначено для автоматизированного подбора состава инструментальной легированной стали при учёте технологических ограничений и эксплуатационных требований. Система помогает определить минимально дорогой состав, обеспечивающий требуемые свойства материала.

Входные данные

  • диапазоны содержания Cr, Ni, Mo и Mn;
  • стоимость легирующих элементов;
  • требования по прочности, твёрдости и температуре плавления;
  • технологические ограничения на сумму добавок и сочетания элементов.

Результат работы

  • рассчитанный состав сплава;
  • оценка прочности, твёрдости и температуры плавления;
  • итоговая стоимость состава;
  • проверка выполнения ограничений и визуализация результата.

Математическая модель

В основе приложения лежит модель, связывающая содержание легирующих элементов Cr, Ni, Mo и Mn со свойствами сплава. Оптимизация выполняется по критерию минимальной стоимости при соблюдении заданных требований и технологических ограничений.

σ = σ₀ + kCr·Cr + kMo·Mo + kCrMo·Cr·Mo
HRC = HRC₀ + kNi·Ni + kMn·Mn + kNiMn·Ni·Mn
T = T₀ − kT·(Cr + Ni + Mo + Mn)

Целевая функция:

C = cCr·Cr + cNi·Ni + cMo·Mo + cMn·Mn → min

Ограничения:

σ ≥ σreq
HRC ≥ HRCreq
T ≥ Treq
summin ≤ Cr + Ni + Mo + Mn ≤ summax
Cr · Ni ≤ crnimax

Здесь Cr, Ni, Mo и Mn — содержание легирующих элементов, %; σ — прочность, МПа; HRC — твёрдость; T — температура плавления, °C; C — стоимость состава; σ₀, HRC₀, T₀ — базовые значения свойств; k — коэффициенты влияния элементов.

Технология работы пользователя

  1. ввод исходных данных и выбор варианта расчёта;
  2. запуск оптимизации;
  3. просмотр результата и проверка ограничений;
  4. визуализация и экспорт в Excel.
